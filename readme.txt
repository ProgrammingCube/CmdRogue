CmdRogue

CmdRogue is a batch file roguelike game. To date, it is the first one that will ever reach maturity.


This is a PRE BETA. THERE WILL BE BUGS

Several include not dying right, monsters not dying right, and weird graphics stuff.

However, this is a demo of something greater to come.




Controls:

Arrow keys to move

Escape or "q" to exit

m for messages

. for wait turn

when prompted for debug mode, type "n".

Have fun!
